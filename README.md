# Agricultural-Crops-Image-Classification-and-Disease-Detection-using-Deep-Learning
This project focuses on identifying crop types and detecting plant diseases using deep learning techniques. It helps farmers and agricultural experts diagnose plant diseases early and take preventive measures to improve crop yield.
1.Features
- Crop image classification using Convolutional Neural Networks (CNN)
- Plant disease detection from leaf images
- Trained deep learning model for accurate predictions
- Lightweight model deployment using TensorFlow Lite
- User-friendly interface for prediction
2.Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy, Pandas
- Deep Learning (CNN)
3.Project Structure
- `main.py` – Main execution file
- `disease.py` – Disease prediction logic
- `model.h5` – Trained deep learning model
- `model.tflite` – Lightweight model for deployment
- `plantdisease.ipynb` – Model training and experimentation
